# CHECKLIST_API551_PDF_FORMATTING_APPROVED

Дата версии: 2026-06-16

Статус: APPROVED / v5-patch по `RULES_API551_PDF_FORMATTING_VERIFIED_v2`

## 0. Полнота информации

- [ ] Не потерян ни один содержательный абзац источника.
- [ ] Не потеряны NOTE.
- [ ] Не потеряны таблицы.
- [ ] Не потеряны сноски.
- [ ] Не потеряны Figures.
- [ ] Все исключённые из PDF элементы являются только служебными слоями или техническими артефактами.
- [ ] Нет формулировок «этот абзац не включать» для основного текста источника.

## 1. Назначение PDF

- [ ] PDF является русской технической печатной версией.
- [ ] PDF не является двухколоночным REVIEW-документом.
- [ ] Нет английской OCR-колонки.
- [ ] Включены все элементы, входящие в финальную сборку: основной текст, разделы, списки, таблицы, рисунки, подписи, примечания, сноски, библиография/содержание/перечни при наличии.

## 2. Потоковая вёрстка

- [ ] Русский текст идёт непрерывным потоком сверху вниз.
- [ ] Не повторяется жёстко постраничная структура исходного английского PDF.
- [ ] Переносы выполняются по русскому тексту.
- [ ] Таблицы, рисунки, подписи, примечания и сноски размещены как связанные объекты.

## 3. Страница, поля, колонтитулы

- [ ] Формат A4.
- [ ] Ориентация портретная по умолчанию.
- [ ] Верхнее поле 1 cm.
- [ ] Нижнее поле 1 cm.
- [ ] Левое поле 2 cm.
- [ ] Правое поле 1 cm.
- [ ] Основной текст не перекрывает верхний колонтитул.
- [ ] Нижний колонтитул не перекрывается текстом, таблицами, рисунками, сносками или подписями.
- [ ] Номер страницы русского PDF идёт по русскому потоку.
- [ ] Номера исходных API/PDF-страниц не выглядят как основная нумерация русского PDF.
- [ ] Альбомная страница не применена автоматически без ручного решения.

## 4. Шрифты

- [ ] Основной текст Times New Roman 12 pt.
- [ ] Основные нумерованные заголовки Times New Roman 14 pt, полужирный.
- [ ] Вложенные технические заголовки 12-14 pt, единообразно.
- [ ] Таблицы 12 pt или 11 pt при необходимости.
- [ ] В таблицах нет шрифта ниже 11 pt без ручного решения.
- [ ] Подписи таблиц 12 pt.
- [ ] Подписи рисунков 12 pt.
- [ ] Сноски 8 pt.
- [ ] Номера сносок 8 pt, надстрочно.
- [ ] Русские символы, градусы, греческие буквы, индексы и спецсимволы отображаются без квадратов.
- [ ] Шрифты встроены или гарантированно корректно отображаются.

## 5. Абзацы, заголовки, списки

- [ ] Выравнивание основного текста применено единообразно.
- [ ] Межстрочный интервал читаемый, ориентир 14-14.5 pt.
- [ ] Нет случайных больших межабзацных пустот.
- [ ] Первый абзац после заголовка связан с заголовком.
- [ ] Нет одиночных висячих строк, если это можно исправить вёрсткой.
- [ ] Нет ручных пустых строк для имитации переносов.
- [ ] Все номера разделов и подразделов сохранены.
- [ ] Заголовки не оставлены внизу страницы без текста.
- [ ] Типы и уровни списков сохранены.
- [ ] Маркер списка не разорван с первой строкой пункта.
- [ ] Вложенные списки имеют корректный висячий отступ.

## 6. Технические переносы

- [ ] Не разорваны число и единица измерения.
- [ ] Не разорваны знак `±` и значение.
- [ ] Не разорваны `рисунок N`, `таблица N`, `раздел N`.
- [ ] Не разорваны обозначения стандартов.
- [ ] Не разорваны диапазоны температур/давлений.
- [ ] Не разорваны химические формулы и марки материалов.
- [ ] Единицы и диапазоны сохранены технически корректно.

## 7. Таблицы

- [ ] Обычные таблицы реконструированы как таблицы, а не вставлены картинкой.
- [ ] Название таблицы размещено над таблицей.
- [ ] Название таблицы и первая строка таблицы не разделены переносом страницы.
- [ ] Таблица не выходит за правую границу печатной области.
- [ ] Текст в ячейках переносится и читается.
- [ ] Структура строк, столбцов и объединённых ячеек сохранена.
- [ ] Числовые значения, размерности и технические обозначения не искажены.
- [ ] Таблица идёт после первого логического упоминания, если помещается.
- [ ] Таблица, помещающаяся на пустой странице, не разорвана.
- [ ] Большая таблица при переносе повторяет шапку.
- [ ] При продолжении есть пометка `Продолжение таблицы N`, если таблица разорвана.
- [ ] Строки таблицы не разорваны до потери читаемости.

## 8. Рисунки / Figure

- [ ] Figure на Stage 1 берётся из approved crop, если есть approved crop.
- [ ] Самодельная переобрезка не применена при наличии approved crop.
- [ ] Рисунок не выходит за печатную область.
- [ ] Рисунок не разорван переносом страницы.
- [ ] Подпись размещена под рисунком.
- [ ] Подпись и рисунок не разделены переносом страницы.
- [ ] В финальной сборке использованы approved-рисунки с переведёнными надписями, если approved-пакет принят.
- [ ] Внутренние содержательные надписи, NOTE, встроенные таблицы и пояснения Figure имеют русскую передачу.
- [ ] Встроенные таблицы внутри Figure реконструированы как таблицы, если они содержательные.
- [ ] Встроенные NOTE внутри Figure оформлены как примечания/таблица примечаний.
- [ ] Figure-абзацы не удалены из-за границ crop.

## 9. Размещение объектов

- [ ] Таблица размещена сразу после абзаца первого упоминания, если помещалась.
- [ ] Рисунок размещён после первого логического упоминания.
- [ ] Если объект перенесён, пустота заполнена только допустимым последующим текстом.
- [ ] Заполнение пустоты остановлено перед стоп-элементом.
- [ ] Перенесённый объект идёт перед дальнейшим текстом, не использованным как заполнитель.
- [ ] Группа объектов сохранена в порядке упоминания.
- [ ] Объект не перенесён в чужой раздел.
- [ ] Раздел-владелец определён по самому глубокому активному нумерованному заголовку.

## 10. NOTE и сноски

- [ ] NOTE переведены как `ПРИМЕЧАНИЕ` или `ПРИМЕЧАНИЕ N` по контексту.
- [ ] Ссылки на NOTE сокращены как `Прим. N`, если это ссылка.
- [ ] NOTE основного текста и NOTE внутри Figure не смешаны.
- [ ] Группы `NOTE 1`, `NOTE 2`, `NOTE 3` не объединены в один абзац.
- [ ] NOTE, относящиеся к объекту, не отделены от него без необходимости.
- [ ] Сноска размещена на странице ссылки.
- [ ] Сноска не перекрывает нижний колонтитул.
- [ ] Если сноска не помещалась, перенесён связанный абзац со ссылкой.
- [ ] Нумерация сносок видимая и соответствует ссылкам.

## 11. Формулы, размеры и единицы

- [ ] Формулы сохранены читаемо.
- [ ] Индексы, степени, греческие буквы и математические знаки не упрощены до потери смысла.
- [ ] Диапазоны значений сохранены полностью.
- [ ] Число и единица связаны неразрывным пробелом.
- [ ] Температуры не разорваны между числом и `°C` / `°F`.
- [ ] Дюймовые обозначения не заменены автоматически внутри числовых размеров, если это искажает источник.
- [ ] В заголовках таблиц единицы переведены только если это не меняет техническую запись.

## 12. Ссылки и аннотации

- [ ] Все внутренние ссылки на рисунки, таблицы, разделы, примечания и сноски являются активными PDF-аннотациями, если цель есть в PDF.
- [ ] Внутренние ссылки ведут на правильные якоря текущего PDF.
- [ ] Нет временных ссылок на исходный PDF для объектов, уже включённых в финальный PDF.
- [ ] Упоминания `рисунок N` и `таблица N` оформлены единообразно.
- [ ] Между словом `рисунок`/`таблица` и номером стоит неразрывный пробел.
- [ ] Нет цветного текста без ссылки, если он должен быть ссылкой.
- [ ] Проверены координаты link annotations.

## 13. Содержание и перечни

- [ ] Содержание, если включено, переведено.
- [ ] Содержание синхронизировано с фактическими страницами русского PDF.
- [ ] Содержание ссылается на страницы русского PDF, а не исходного API.
- [ ] Перечни рисунков и таблиц имеют рабочие ссылки на объекты.
- [ ] При изменении пагинации содержание и перечни пересобраны.

## 14. Запрещённое служебное содержимое

- [ ] Нет левой OCR/английской колонки.
- [ ] Нет raw OCR.
- [ ] Нет служебных OCR-блоков.
- [ ] Нет JSON.
- [ ] Нет HTML/CSS/JS-кода.
- [ ] Нет кнопок, фильтров, элементов поиска, навигации HTML-интерфейса.
- [ ] Нет отчётов проверки внутри PDF.
- [ ] Нет debug/TODO/REBUILD/служебных маркеров.
- [ ] Нет внутреннего анализа Figure как служебного JSON/отчёта.

## 15. Проверка после генерации

- [ ] PDF открывается без ошибок.
- [ ] Все страницы отрендерены в изображения для проверки.
- [ ] Контрольные страницы визуально проверены.
- [ ] Проблемные страницы вокруг перенесённых объектов проверены.
- [ ] Текст, таблицы, рисунки, подписи и сноски не перекрываются.
- [ ] Нет обрезанного текста.
- [ ] Нет объектов за правой границей.
- [ ] Проверены link annotations.
- [ ] Проверены шрифты/глифы.
- [ ] Проверено отсутствие служебного содержимого.
- [ ] Количество страниц зафиксировано в кратком отчёте.

## 16. Критические условия браковки

- [ ] Нет потерянного содержательного абзаца, NOTE, таблицы, сноски или Figure.
- [ ] Нет обрезанного текста или таблиц.
- [ ] Нет объектов за печатной областью.
- [ ] Нет отделённой подписи от рисунка/таблицы.
- [ ] Нет сноски не на странице ссылки.
- [ ] Нет сноски, перекрывающей колонтитул.
- [ ] Нет массово неработающих внутренних ссылок.
- [ ] Нет цветных «ссылочных» упоминаний без аннотаций.
- [ ] Нет raw OCR, JSON, HTML-кода или debug-маркеров.
- [ ] Поля и базовый шрифт корректны.
- [ ] Нет переобрезанного не по approved crop рисунка без ручного решения.
- [ ] Обычная таблица документа не вставлена картинкой вместо реконструкции.
- [ ] Figure/таблица не перенесены в чужой раздел.
