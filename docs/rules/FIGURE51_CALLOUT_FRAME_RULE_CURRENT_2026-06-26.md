# Figure 51 Callout Frame Rule

Status: CURRENT explicit user-approved Figure 51 rule.

Figure 51 leader-line callouts use a gray outline/frame only. Do not use solid gray fill behind the callout text. The callout interior remains white/transparent except for safely cleared source text and rendered Russian text.

The gray frame remains compact, text-driven, and touching the corresponding leader line. The frame side adjacent to the leader line is fixed; expansion is allowed only away from the leader line.

Figure captions are not leader-line callouts and stay outside this gray-frame treatment unless a later explicit figure-specific rule reopens that decision.
